# Stock Market Predictor Frontend

This is a React-based frontend for the Stock Market Predictor project.

## Features
- Clean, modern UI
- Prediction page with input for stock symbol
- Submit button and result display
- Uses dummy data for predictions (ready to connect to backend)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view the app.
